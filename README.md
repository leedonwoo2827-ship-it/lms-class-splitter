# LMS 입과명단 분반기

여러 회사의 LMS 입과 엑셀을 한 번에 받아 **과정별로 모은 뒤 정원(기본 50명) 기준으로 자동 분반**해 주는 Windows GUI 도구입니다.

회사 LMS에서 정원 초과로 입과되는 사고를 막고, 강의 개설 단위(50명/반)에 맞춰 깔끔하게 정리된 엑셀을 즉시 받아볼 수 있습니다.

---

## 다운로드 (설치 불필요)

> 더블클릭 한 번이면 실행됩니다. Python 설치 필요 없음.

**[▶ LMS입과명단분반기.exe 다운로드](https://github.com/leedonwoo2827-ship-it/lms-class-splitter/raw/main/LMS입과명단분반기.exe)**

- 파일 크기: 약 34MB
- 동작 환경: Windows 10 / 11

> Windows SmartScreen 경고가 뜨면 **추가 정보 → 실행** 으로 진행하세요. PyInstaller로 패키징한 검증되지 않은 실행파일이라 발생하는 일반적인 경고입니다.

---

## 사용법

1. **파일 추가** — "파일 추가..." 또는 "폴더 추가..." 버튼으로 LMS 입과명단 엑셀들을 불러옵니다.
2. **헤더 확인** — "헤더 미리보기" 버튼으로 각 파일의 컬럼명을 확인할 수 있습니다.
3. **옵션 설정**
   - **과정 컬럼명** — 과정명이 들어있는 헤더 이름. 기본값 `입과과정` (다르면 수정).
   - **반 정원** — 기본 `50` 명.
   - **출력 폴더** — 비워두면 입력 파일 옆 `_output/` 자동 생성.
4. **분반 실행** — 클릭하면 과정별로 `{과정명}_1반.xlsx`, `{과정명}_2반.xlsx` … 형태로 저장됩니다.

---

## 동작 방식

- 입력된 모든 엑셀의 행을 첫 파일의 컬럼 구조로 통합
- 지정한 **과정 컬럼**의 값으로 그룹핑
- 각 그룹을 정원만큼 잘라 별도 엑셀(시트 1개)로 저장
- 헤더 행 1개 + 데이터 행으로 구성된 깔끔한 출력 → 다른 시스템으로 import 가능

## 소스에서 직접 빌드하기

Python 3.10+ 환경에서:

```bash
git clone https://github.com/leedonwoo2827-ship-it/lms-class-splitter.git
cd lms-class-splitter
pip install -r requirements.txt
python app.py
```

단일 exe로 패키징하려면:

```bash
build_exe.bat
```

`dist\LMS입과명단분반기.exe` 가 생성됩니다.
